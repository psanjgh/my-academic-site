## Responsibilities

**Physics 1A: Foundations (PHYS08016)** — Assisted with foundational physics workshops covering kinematics, Newtonian mechanics, work and energy, linear momentum, angular momentum, oscillations, and waves.

**Computer Simulation (PHYS08026)** — Led workshops on key concepts of object-oriented programming in Python, analysis of problem statements to produce simple OO designs, use of scientific programming and visualization packages, and writing graphical applications to visualize experimental results.

**Computer Modelling (PHYS09057)** — Guided students in understanding computer simulation and modelling as part of scientific methodology, basic code and algorithm design, software engineering best practices, simple OOP in Python, and use of external tools (VMD) for analyzing modelling results.
